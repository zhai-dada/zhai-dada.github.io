误码仪学习

前言

通信系统是用以完成信息传输过程的技术系统的总称。在历史上，信息从A到B有很多方式，比如狼
烟，邮寄，电报，电话，互联网，WIFI/BT，卫星通信，光纤，未来技术等等，都是通信系统。都是由信源、信道、信宿三者组成。

graph LR
    A[信源] ---> B[信道] ---> C[信宿]

那么如何衡量一个通信系统是好还是坏呢？我认为主要是从可靠性以及速率上来看。速率也就是bps，单位时间内传输数据越多越好。可靠性主要就是看误码率，眼图质量，抖动，信噪比。

在现代通信系统中，数据传输的准确性至关重要。误码率（BER，Bit Error Rate）是衡量通信链路质量的关键指标，而误码仪（BERT，Bit Error Rate Tester）是用于测试和分析误码率的重要仪器。

误码率可以理解成为信源发送已知数据，经过信道到信宿，信宿对比接收到的数据和已知的发送数据来计算误码率。误码仪可以当作是信源和信宿（分为发送端PPG和接受端ED）能够测试出速率以及误码率。

什么是误码仪

误码仪（BERT）是一种用于测试数字通信系统误码率的设备，它可以产生和接收特定的数据序列，并对比接收到的数据与发送的数据是否一致，以此来计算误码率。误码仪广泛用于验证信号传输质量，确保通信链路达到预期性能。

误码仪的应用场景

关于误码仪，了解了两天，对一些基础的信息进行了学习，有待更深入的研究学习。

比如用在常见的高速PCB板卡，各个高速连接器产品，高速射频线缆，各种高速芯片，整体系统的测试等等。

误码仪应用关键字

  • PPG：脉冲码型发生器，负责发送不同速率不同码型的高速信号
  • ED：误码检测器，接收到信号后，和自己的参考信号一一比对，确定传输的误码个数和误码率
  • CDR：时钟恢复器，从数据中恢复时钟
  • BER：误码率，是衡量数据在规定时间内数据传输精确性的指标，
  • Eye Diagram：眼图
  • Baud Rate：波特率，每秒多少个符号。NRZ下1符号=1比特，PAM4下1符号=2比特
  • Data Rate：比特率，每秒传输的bit数量
  • Jitter：抖动，信号跳变沿偏离理想时间的偏差，该变的时候没变，不该变的时候变了
    • RJ (Random Jitter)：随机抖动，服从高斯分布，无界，主要由热噪声等随机过程引起。
    • PJ (Periodic Jitter)：周期性抖动，按固定周期重复，通常由串扰、电源纹波等产生。
    • SJ (Sinusoidal Jitter)：正弦抖动，PJ 的一种特殊情况，纯正弦波调制形式。
    • BUJ (Bounded Uncorrelated Jitter)：有界不相关抖动，有界的随机抖动，不与数据相关，通常模型化为均匀分布，常见于串扰场景。
  • ISI：码间干扰，前一个比特污染后一个比特，频率越高、传输距离越长就越严重
  • DUT：被测设备
  • SNR：信噪比， ：信号的有效功率，：噪声的有效功率
  • UDP：用户自定义码型

误码仪的用途

误码仪的用途：对数字信号进行高速、高精度的误码率测试，

PAM4和NRZ编码

PAM4（4-Level Pulse Amplitude Modulation）即四电平脉冲幅度调制，PAM4信号技术是一种采用4个不同的信号电平来进行信号传输的调制技术。

NRZ信号采用高、低两种信号电平表示数字逻辑信号的1、0，每个周期可以传输1bit的逻辑信息；
PAM4信号则采用4个不同的信号电平进行信号传输，即00、01、10、11，每个周期可以传输2bit的逻辑信息。因此，在相同符号周期内，PAM4信号的比特速率是NRZ信号的两倍；

PAM4的优势劣势以及应用

PAM4：一个符号两个bit，在同一带宽下传输两倍数据量。核心优势就是在同样的带宽下有着更高的速率。但是SNR要求比NRZ高约9.6dB，误码率要求2.2e-4，FEC（前向纠错编码）变成必要条件，研发/生产成本增大。应用场景就是高速以太网、PCIe6.0等等。

PRBS码

PRBS：Pseudo-Random Binary Sequence，中文翻译叫做伪随机二进制序列，简称伪随机码。主要是测试误码率的情况，例如我们常用的一些协议，PCIE、USB、以太网或者下图的光模块的测试等等。

为什么业界公认选择这种prbs码型进行测试（仿真）呢？主要原因是这种码型与真实链路的数据传输情况非常接近。因为在真实情况中，所以的数据组合都是随机出现的，没有任何规律可言。

而PRBS的码流在很大程度上具有这种“随机数据”的特性，“0”和“1”随机出现，这种码流的频谱特征和白噪声非常接近，所谓“白噪声”就是在一个比较宽的频域里功率密度谱均匀分布，也就是所有的频率都具有相同的能量，因此该码型能够模拟各种不同频率数据组成的情况，使测试更符合真实的情况。

对于眼图测试、误码率和抖动容限测试，最常用的测试码是PRBS，主要有PRBS7、PRBS15、PRBS23和PRBS31。

眼图

眼图，是由于示波器的余辉作用，将扫描所得的每一个码元波形重叠在一起，从而形成眼图。

观察眼图的方法是:用一个示波器跨接在接收滤波器的输出端，然后调整示波器扫描周期，使示波器水平扫描周期与接收码元的周期同步，这时示波器屏幕上看到的图形像人的眼睛，故称 为“眼图”。

信号不可能每次高低电平的电压值都保持完全一致，也不能保证每次高低电平的上升沿、下降沿都在同一时刻，所以由于多次信号的叠加，眼图的信号线变粗，出现模糊(Blur)的现象。所以眼图也反映了信号的噪声和抖动。

眼图的分析

在纵轴电压轴上，体现为电压的噪声(Voltage Noise)；
在横轴时间轴上，体现为时域的抖动(Jitter)。

当存在噪声时，噪声将叠加在信号上，观察到的眼图的线迹会变得模糊不清。若同时存在码间串扰，“眼睛”将张开得更小。一般眼图的眼睛睁得越大，眼图眼高越高，代表信号质量越好。

由眼图可以观察信号间干扰和噪声的影响，具体描述如下：

  • 眼张开度（Eye opening），反映了信号上的噪声，即眼图的“眼睛”张开的大小反映了信号间串扰的强弱。“眼睛”张开越大，且眼图越正，表示信号间串扰越小，反之表示信号间串扰越大。
  • 上冲和下冲，是指信号第一个峰值和谷值和下一个峰值和谷值，由于信号路径的中断产生的峰值失真，主要表现为一个尖端脉冲，过分的过冲能导致电路元器件的失效。
  • 眼交叉比代表着信号的抖动，抖动如果很大就会造成误码率（BER）增加，反之则代表信号质量越好，误码率（BER）越低。
  • 眼图闭合度（Eye closure），即眼顶、眼底和转换区域宽表明接收器判断灵敏度降低。因为噪声将叠加在信号上，观察到的眼图的线迹会变得模糊不清。

眼图的意义

眼图，个人理解就是直观地看信号质量的工具。

带宽

带宽是指信号所占据的频带宽度；

  1. 在被用来描述信道时，带宽是指能够有效通过该信道的信号的最大频带宽度。
  2. 对于模拟信号而言，带宽又称为频宽，以赫兹（Hz）为单位。例如模拟语音电话的信号带宽为3100Hz（从300Hz到3400Hz）。
  3. 对于数字信号而言，带宽是指单位时间内链路能够通过的数据量。例如ISDN的B信道带宽为64Kbps。由于数字信号的传输是通过模拟信号的调制完成的，为了与模拟带宽进行区分，数字信道的带宽一般直接用波特率或符号率来描述。

带宽 = 最高频率 - 最低频率

  • 一个信道的频率范围是 1000Hz 到 6000Hz，那么它的带宽就是 6000 - 1000 = 5000Hz
  • 这就好比一条公路，只允许发动机转速在1000转/分到6000转/分的卡车通行。

根据通信领域的奈奎斯特-香农定理，在一个理想的无噪声信道中：最高码元速率 = 2 × 带宽

为什么是2倍？ 一个带宽为B的通道，最多可以每秒完整地“描绘”出2B个独立的信号变化（码元）。这有点像电影帧率，你需要至少两倍的采样率才能还原一个波形。

带宽 = 公路宽度
码元速率 = 每小时能通过收费站的卡车数量

一条单车道的窄路（低带宽），每小时能通过的卡车数量（码元速率）必然有限。而一条八车道的高速公路（高带宽），单位时间内能通过的卡车数量就多得多。

什么是均衡技术

衰减

衰减就是信号经过信道之后，因为损耗造成信号幅度减弱，能量减弱的现象。衰减本身不会让信号变形，让信号变形的是：不同频率的信号衰减的程度不一致，从而导致了ISI码间干扰。
真正影响信号可靠传输的不是衰减的本身，而是信道衰减随着频率的变化。
也就是说高频信号和低频信号经过信道的衰减不一样，导致高频低频信号形成衰减差，从而导致ISI码间干扰。
信道对信号有一个低通滤波的作用，高频信号衰减的比较大，因此加入均衡，相当于人为地补偿高频信号和低频信号的衰减差值，在发送端添加相当于添加了一个高通滤波器。
但是信号幅度过小会导致SNR不足、误码率上升。平坦衰减容易补偿，比如直接加上增益。

  • TX 端：FFE（前馈均衡）
  • RX 端：FFE（前馈均衡）、CTLE（连续时间线性均衡）、DFE（判决反馈均衡），高通补偿 + 从时域直接减ISI尾巴

均衡

均衡真正解决并不是信号的衰减，而是高低频信号的衰减差。因为信道对信号具有低通的效应，也就是高频信号衰减要比低频信号的衰减要大，因此均衡分为两个思路：

  1. 增强高频分量信号来补偿高频信号在信道中的衰减强的情况，抬高高频分量，也就是预加重。
  2. 减弱低频分量信号来补偿低频信号在信道中的衰减弱的情况，压低低频分量，也就是去加重。

实际中更常用去加重，因为发送端摆幅有上限（电源电压限制了），高频放大不了太多。所以工程上全是压低低频实现的。

均衡类型和具体实现

FFE

前端均衡，常用于高速信号传输的发送端或接收端。

  • 作用：在发送前对信号预畸变，补偿信道低频损耗
  • 原理：FIR 滤波器

CTLE

连续时间线性均衡器，常用于RX接收端

  • 作用：补偿信道高频衰减，把眼图重新睁开
  • 原理：一个高通增益放大器，低频增益低、高频增益高，精确抵消信道的高低频衰减差。缺点是噪声也一起放大

DFE

判决反馈均衡器，是非线性的，用于RX接收端

  • 作用：消除已判 bit 对当前 bit 的 ISI 拖尾
  • 原理：判决器先拍板当前 bit 是 0/1 → 根据前面 N 个已判 bit 的值 × 各自拖尾系数 → 从当前信号里直接减去这些拖尾量。只减 ISI，只放大高频信号，不放大噪声，但前一个判错后面全错

DFE的本质是降低了码间干扰。

抖动

抖动是高速串行设计中的最关键的因素，BERT的测试中离不开抖动，PPG的输出抖动越小越好，PPG最好能够插入高精度的抖动分量。

Jitter：抖动，信号跳变沿偏离理想时间的偏差，该变的时候没变，不该变的时候变了

  1. RJ (Random Jitter)：随机抖动，服从高斯分布，无界，主要由热噪声等随机过程引起。
  2. PJ (Periodic Jitter)：周期性抖动，按固定周期重复，通常由耦合到系统中的外部确定噪声源引起的。
  3. SJ (Sinusoidal Jitter)：正弦抖动，PJ 的一种特殊情况，纯正弦波调制形式。
  4. BUJ (Bounded Uncorrelated Jitter)：有界不相关抖动，有界的随机抖动，不与数据相关，通常模型化为均匀分布，常见于串扰场景。
  5. DCD，占空比失真抖动。
  6. SSC，扩频时钟。

RJ

随机抖动，服从高斯分布，无界，主要是有热噪声等随机过程引起的，比如VCO随机抖动，半导体热噪声。

PJ

周期性抖动，按固定周期重复，周期性抖动一般是由耦合到系统中的外部确定噪声源引起的。

SJ

正弦抖动，属于是PJ的一种特殊情况，是纯正弦波调制产生的。

BUJ

有界不相关抖动，主要是高密度布线之间，线与线之间的串扰引发的。

DCD

占空比失真抖动，比特信号的高低持续时间不相等，变得偏长或者偏短。是DJ的一种。

SSC

把时钟能量分散到一段范围之内，比如100M时钟分散到99.5M - 100.5M，以此来降低EMI（电磁干扰）

串扰

什么是串扰

串扰就是两条信号线之间的耦合，信号线之间的互感、互容引发的信号线上出现噪声。

串扰的影响

  1. 串扰引发的数字电平错误，比如由串扰脉冲引发相邻线上产生噪声脉冲从而导致电平错误。
  2. 串扰引发的时许错位，比如串扰发生在跳变沿上，从而导致时许的提前或延后。

如何消除串扰

  1. 增大线距
  2. 加强隔离
  3. 差分走线
  4. 减少平行走线长度
  5. 加入纠错手段

抖动容限、噪声容限

  • 抖动容限：抖动容限指系统在保持额定性能水平的情况下，能够容忍的时序抖动量。抖动容限是衡量通信系统或设备在不导致误码率（BER）显著增加的情况下，能够承受输入信号中时序抖动的最大幅度。是一个重要的指标，用于表征通信系统或设备接口在输入抖动情况下的性能。
  • 噪声容限：噪声容限是系统在信号受到干扰或电压波动时仍能正确识别逻辑状态的能力。

前者看判得准不准，后者看分不分得清。

CMI

噪声同相位叠加到差分线上，二者抵消，但是二者不平衡会造成信号损伤，只能使用走线对称或者其他的手段来解决。

DMI

噪声反相位叠加到差分线上，造成信号损伤。可以使用DFE/CTLE来改善。

PSIJ

电源噪声。板级/模块DC-DC纹波及谐振通常在500KHz-20MHz，绕过CDR低通滤波，直接压
缩采样裕量。
